# In memory database

## Setup

Make sure you have node installed

1. Install dependencies (@types/node/linting)

```
npm install
```

*Make sure you run this command from the root folder where the package.json is located*
## Run

To run the program just execute 

```
npm run start
```

*Make sure you run this command from the root folder where the package.json is located*
